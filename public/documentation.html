Movie API

This is an api that allows users to retrieve movie data such as name of the movie, genre, directors etc. Users will also be able to register and edit their favorite movies list or deregister.

Request URL HTTP Method URL parameters Request body data format Respond body data format
Get a list of ALL movies /movies GET None A JSON object holding data about all movies
Get data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title /movies/[Title] GET /movies/:title None A JSON object holding data about a single movie containing a title, description, genre, director, image and featured.

Example:
{ "title": "How to Marry a Millionaire", "description": "How to Marry a Millionaire is a 1953 American romantic comedy film. ", "director": "Jean Negulesco", "genre":"Romantic comedy", "image": "https://images-na.ssl-images-amazon.com/images/I/41MKEDHVBTL._SY445_.jpg", "featured": true }
Get data about a movie genre by name /movies/genres/[Genre] GET /movies/genres/:genre None A JSON object holding data about the genre
Get data about a movie director by name /movies/directors/[Name] GET /movies/directors/:name None A JSON object holding data about the director
Register new user /users POST A JSON object holding data about the user to create.
Example:
{ “name” : “John Doe", “Username”: “johndoe”}		 A JSON object holding data about the student that was added.
Example:
{ “name” : “Jane Doe", “Username”: “janedoe”}
Update a user information /users/[Username] PUT /users/:username None A text message with the info that was updated.
Add a movie to Favourites /users/[Username]/favourites POST /users/:username/favourites A JSON object holding data about the movie to add to favourites.
Example:
{ “title” : “How to Marry a Millionaire"}		 A JSON object holding data about the movie that was added to favourites.
Example:
{ “title” : “How to Marry a Millionaire"}
Remove movie from Favourites /users/[Username]/favourites/[Title] DELETE /users/:username/favourites/:title None A message indicating whether the movie was successfully removed from favourites.
Unregister an existing user /users/[Username] DELETE /users/:username None A text message indicating whether the user was successfully unregistered
